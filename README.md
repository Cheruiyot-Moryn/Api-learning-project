# Api-learning-project
Learning REST APIs using Postman
📌 Overview

This project explores REST APIs and how applications communicate using HTTP requests and JSON responses.

I used JSONPlaceholder for learning basic API requests and OpenWeather API for real-world API interaction.

🎯 Objectives
Understand how APIs work
Send GET and POST requests
Interpret JSON responses
Test APIs using Postman

🛠️ Tools Used
Postman
JSONPlaceholder API
OpenWeather API

🚀 How to Run / Use
Open Postman
Use a GET request:
https://jsonplaceholder.typicode.com/posts
Click "Send"
View the JSON response

📂 Project Structure
toolkit.md → Full documentation
screenshots/ → API request & response images

📖 Key Learnings
APIs use HTTP methods like GET and POST
Responses are usually in JSON format
API keys are required for some APIs

⚠️ Challenges Faced
Understanding API responses at first
Fixing errors with incorrect URLs
Handling API key issues
